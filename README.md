## 快速开始

### 环境要求

- Python >= 3.11   - Python >= 3.11   - Python >= 3.11   - Python >= 3.11
- [uv](https://docs.astral.sh/uv/) 包管理器

### 安装

```bash
git clone https://github.com/1308495420-dotcom/Coding-Agent.git
cd Coding-Agent   cd 编码代理   cd 编码代理

# 安装依赖
uv sync

# 配置 LLM API
cp .mewcode/config.yaml.example .mewcode/config.yaml
# 编辑 .mewcode/config.yaml，填入你的 API Key 和模型
```

### 运行

```bash
# 交互式 TUI 模式
uv run mewcode

# 非交互模式（单次提问）
uv run mewcode -p "你的问题"

# 远程浏览器模式
uv run mewcode --remote
# 浏览器打开 http://localhost:18888
```
